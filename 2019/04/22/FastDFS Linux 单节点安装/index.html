  • 2019年04月22日
  • duyu

备注: mac安装由于SIP(System Integrity Protection, 系统完整性保护功能), 无法在/usr/目录下创建文件, 所以需要关闭SIP

  • 重启 Mac, 按住 Command + R, 进入 recovery 模式
  • 选择打开 Utilities下的终端, 输入 csrutil disable 并回车
  • 重启 Mac 即可

1.准备

  • libfastcommon: 从 FastDFS和FastDHT中提取出来的公共C 函数库，基础环境 https://github.com/happyfish100/libfastcommon.git
  • fastdfs: 主程序 https://github.com/happyfish100/fastdfs.git
  • fastdfs-nginx-module: nginx扩展支持模块 https://github.com/happyfish100/fastdfs-nginx-module.git
